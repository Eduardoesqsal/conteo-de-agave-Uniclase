INSTRUCCIONES PARA EJECUTAR EL SCRIPT

1. Abrir la consola de comandos
- Presiona las teclas Windows + R
- Escribe: cmd
- Presiona Enter

2. Ir a la carpeta del proyecto
- Abre el Explorador de archivos
- Ubica la carpeta del proyecto (está en el Escritorio)
- Copia la ruta que aparece en la barra de direcciones
- En la consola escribe:
  cd RUTA_DE_LA_CARPETA
- Presiona Enter

Ejemplo:
cd C:\Users\TuUsuario\Desktop\NombreDelProyecto

3. Crear el entorno virtual (solo la primera vez)
- Ejecuta el siguiente comando:
  python -m venv venv

4. Activar el entorno virtual
- Ejecuta:
  venv\Scripts\activate

- Si se activó correctamente, aparecerá (venv) al inicio de la línea de la consola

5. Instalar las dependencias
- Asegúrate de que el archivo requirements.txt esté en la carpeta del proyecto
- Ejecuta:
  pip install -r requirements.txt

6. Ejecutar el script
- Ejecuta el archivo Python con:
  python count.py
